# My Profile

## Overview

The My Profile feature allows athletes and parents to view and manage their personal information, emergency contacts, and family relationships. Keep your profile up to date to ensure coaches and admins can reach you in case of emergencies and that your information is accurate for registration and competitions.

## Features

### Profile Information

**Athletes:**
- Full name
- Email address
- Birth date
- Emergency contact name and phone
- Parent/guardian linking
- View linked parent information

**Parents:**
- Full name
- Email address
- Emergency phone number
- View linked athletes
- Manage family relationships

### Emergency Contacts
- Required contact name
- Required phone number
- Validated phone format
- Quick access for coaches in emergencies
- Update anytime

### Family Linking
- Athletes link to parents/guardians
- Parents see their linked athletes
- Secure relationship management
- Privacy-protected information sharing

     - Validated for proper format

4. **Save Changes**
   - Tap **"Save"** button
   - Changes sync to your account

## Profile Fields Explained

### Personal Information

#### Name
- **Purpose**: Identity verification, communications, competitions
- **Format**: First and last name required
- **Changes**: Contact admin for official name changes
- **Display**: Shows throughout app

#### Email
- **Purpose**: Account identification, communication
- **Format**: Valid email address required
- **Login**: Used for Google Sign-In authentication
- **Changes**: May affect sign-in, verify before changing

#### Birth Date (Athletes Only)
- **Purpose**: Age verification for competitions and insurance
- **Format**: MM/DD/YYYY
- **Accuracy**: Must be accurate for competition eligibility
- **Changes**: Contact admin if incorrect, requires verification

### Emergency Contact Information

#### Contact Name
- **Who**: Parent, guardian, or designated emergency contact
- **Required**: Must have someone listed
- **Relationship**: Can note relationship (e.g., "Mother", "Father", "Guardian")
- **Availability**: Should be reachable during practice hours

#### Contact Phone
- **Format**: Standard US phone number
  - (123) 456-7890
  - 123-456-7890
  - 1234567890 (converts to formatted)
- **Required**: Must provide valid number
- **Validation**: App validates format before saving
- **Testing**: Verify number is correct and reachable

### Family Relationships

#### Parent-Athlete Links
- **Security**: Admin-managed to prevent unauthorized access
- **Benefits**: 
  - Streamlined communication
  - Shared calendar access
  - Emergency coordination
  - Profile visibility
- **Privacy**: Only linked family members see each other's info

## Best Practices

### Keep Information Current

**Update Immediately When:**
- Phone number changes
- Email address changes
- Emergency contact changes
- Moving to new address
- Contact person changes

**Regular Reviews:**
- Check profile monthly
- Verify accuracy before season
- Update before competitions
- Confirm after any family changes

### Emergency Contact Guidelines

**Choose Contact Who:**
- Is available during practice times
- Can reach gym quickly if needed
- Is authorized to make medical decisions
- Has reliable phone service
- Knows athlete's medical information

**Multiple Contacts:**
- Consider listing parent as primary
- Have backup contact information
- Provide to coach separately if needed
- Update if availability changes

### Privacy & Sharing

**Keep Private:**
- Don't share login credentials
- Protect email/password
- Don't post profile info publicly
- Verify who you're sharing with

**Share When Appropriate:**
- Coaches need accurate info
- Admins manage system access
- Parents of linked athletes
- Competition organizers (through admin)

## Troubleshooting

### Can't Edit Profile

**Check Sign-In:**
- Must be signed in to edit
- Verify correct account
- Sign out and back in if needed

**Permission Issues:**
- Can only edit your own profile
- Parents can view but not edit athlete profiles directly
- Contact admin for locked fields

**App Issues:**
- Restart app
- Check internet connection
- Update app if available

### Phone Number Not Validating

**Format Requirements:**
- Use standard US format
- Include area code
- Try different formats:
  - (555) 123-4567
  - 555-123-4567
  - 5551234567

**Remove Extra Characters:**
- No letters
- No special characters except () and -
- No country codes (app assumes US)
- No extensions

**Still Not Working:**
- Try entering just numbers
- Let app auto-format
- Contact admin if persistent issue

### Email Changes Not Saving

**Verify Email Format:**
- Valid email address (user@domain.com)
- No spaces
- Proper @ and domain
- Check for typos

**Account Considerations:**
- Email used for Google Sign-In
- Changing email may affect login
- Verify new email before saving
- May need to sign in with new email

**Contact Admin:**
- Email changes may require verification
- Admin can help with account email updates
- Prevents unauthorized account changes

### Family Links Not Showing

**Verify Relationship:**
- Admin must set up parent-athlete links
- Links aren't automatic
- Contact admin to establish relationship

**Check Account:**
- Signed into correct account
- Athlete and parent using right emails
- Both accounts active

**Sync Issues:**
- Pull down to refresh
- Sign out and back in
- Check internet connection

## Need Help?

### Common Questions

**Q: How do I change my birth date?**
A: Contact an admin. Birth dates are restricted to prevent errors and ensure competition eligibility accuracy.

**Q: Can I link multiple athletes as a parent?**
A: Yes, parents can be linked to multiple athletes. Contact admin to set up additional links.

**Q: What if my emergency contact changes?**
A: Edit your profile immediately to update emergency contact name and phone number.

**Q: Can I use a parent as my profile login?**
A: Athletes should have their own accounts. Parents have separate accounts linked to their athletes.

**Q: Why can't I edit certain fields?**
A: Some fields are admin-controlled for security and accuracy. Contact admin for changes.

### Contact Support
If you need help:
1. Verify you're signed into correct account
2. Check that you have internet connection
3. Try signing out and back in
4. Contact coach or admin for:
   - Birth date corrections
   - Family linking setup
   - Profile access issues
   - Locked or missing fields
