# Messages & Announcements

## Overview

The Messages & Announcements feature keeps you connected with important communications from BPVA coaches and administrators. Receive notifications about practice schedules, competition updates, gym closures, and other important information.

## Features

### Message Types
- **Info** (Blue): General information and updates
- **Warning** (Orange): Important notices that need attention
- **Urgent** (Red): Critical messages requiring immediate action
- **Announcement**: Special announcements from coaches

### Notification System
- **Push Notifications**: Receive alerts on your device when new messages arrive
- **Unread Badge**: See number of unread messages on the menu icon
- **Read Status**: Messages are marked as read when you view them
- **Dismissal**: Remove messages from your view after reading

## How to Use

### Accessing Messages

1. **Open Messages Screen**
   - Tap the hamburger menu (☰) in the top-left
   - Select **"Messages"**
   - View your message list

2. **Check Unread Count**
   - The Messages menu item shows unread message count
   - Red badge displays number of unread messages
   - Badge disappears when all messages are read

### Reading Messages

1. **View Message List**
   - Messages are sorted by date (newest first)
   - Unread messages have:
     - Blue highlighted background
     - Bold title text
     - "NEW" badge
   
2. **Open Message Details**
   - Tap on any message to view full content
   - Message is automatically marked as read
   - View sender name and timestamp
   - See message type indicator (color/icon)

3. **Message Information**
   - **Title**: Brief subject line
   - **Content**: Full message text
   - **From**: Sender name (coach or admin)
   - **Type**: Message priority level
   - **Time**: When message was sent

### Managing Messages

#### Mark Messages as Read
- Messages are automatically marked as read when opened
- Read messages have normal background color
- Title text changes from bold to normal

#### Dismiss Messages
1. Tap the **X** icon on the right side of any message
2. Confirm dismissal
3. Message is removed from your view
4. You won't see the message again

#### Mark All as Read
- Available when you have multiple unread messages
- Clears all unread badges at once
- Useful when catching up on messages

## Understanding Message Types

### Info Messages (Blue)
- General announcements
- Practice schedule updates
- Non-urgent information
- **Example**: "Practice this Saturday at 10 AM"

### Warning Messages (Orange)
- Important notices
- Changes requiring attention
- Safety information
- **Example**: "Gym will close early Friday at 8 PM"

### Urgent Messages (Red)
- Critical information
- Immediate action needed
- Emergency notifications
- **Example**: "Practice cancelled due to weather"

## Push Notifications

### Enabling Notifications

**iOS:**
1. Open device Settings
2. Navigate to **BPVA Calendar**
3. Tap **Notifications**
4. Enable **Allow Notifications**
5. Choose notification style (Banner, Alert)

**Android:**
1. Open device Settings
2. Navigate to **Apps** → **BPVA Calendar**
3. Tap **Notifications**
4. Enable notifications
5. Select notification importance

### Notification Behavior
- Notifications appear when new messages arrive
- Sound/vibration based on device settings
- Notification shows message title and preview
- Tap notification to open app and view message

## Troubleshooting

### Not Receiving Messages

**Check Internet Connection:**
- Messages require internet to sync
- Connect to Wi-Fi or cellular data
- Pull down to refresh message list

**Verify Account Status:**
- Sign out and sign back in
- Refresh your profile
- Contact coach if issue persists

### Not Receiving Push Notifications

**Enable Notifications:**
- Check device notification settings
- Enable BPVA Calendar notifications
- Ensure "Do Not Disturb" is off

**Foreground vs Background:**
- App must be allowed to send notifications in background
- Check battery optimization settings
- Ensure app isn't force-stopped

### Messages Appear and Disappear

**Dismissed Messages:**
- Once dismissed, messages don't reappear
- Check with coach if you need message resent
- Dismissal is permanent per user

## Best Practices

### Stay Informed
- Check messages regularly (daily recommended)
- Enable push notifications for urgent updates
- Don't dismiss messages until fully read
- Take action on urgent/warning messages promptly

### Message Organization
- Read new messages daily
- Dismiss old messages to keep list clean
- Mark all as read periodically
- Pay attention to message type colors

### Communication
- Don't reply to messages in app (one-way communication)
- Contact coaches directly for questions
- Use email/text for responses if needed
- Messages are for announcements, not conversations

## Privacy & Security

### Message Visibility
- You only see messages sent to:
  - All users
  - Athletes (if you're an athlete)
  - Parents (if you're a parent)
  - Specific individuals (when you're selected)

### Message Storage
- Messages stored securely in Firebase
- Only authenticated users can access
- Read/dismissed status is private to you
- Message history maintained for reference

## Tips for Athletes & Parents

### Athletes
- Check messages before practice
- Note schedule changes immediately
- Don't miss urgent safety messages
- Keep notifications enabled

### Parents
- Monitor linked athlete messages
- Stay informed about schedule changes
- Note important deadline messages
- Forward relevant info to athletes

## Need Help?

If you're not receiving messages or have issues:
1. Verify you're signed in
2. Check internet connection
3. Review notification settings
4. Sign out and back in
5. Contact a coach or administrator
