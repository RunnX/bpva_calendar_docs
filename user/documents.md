# Documents & Forms

## Overview

The Documents & Forms feature allows athletes and parents to submit required registration documents, waivers, and forms digitally. Upload documents using your device's camera or file picker, track submission status, and receive notifications when documents are reviewed. No more lost paperwork!

## Features

### Document Submission
- **Camera Upload**: Take photos of forms directly in app
- **File Upload**: Select PDFs or images from your device
- **Multiple Formats**: Supports PDF, JPG, PNG image files
- **High Quality**: Camera captures at 85% quality for clarity
- **Instant Upload**: Documents upload immediately to secure cloud storage

### Status Tracking
- **Pending** (Yellow): Document submitted, awaiting review
- **Approved** (Green): Document accepted by admin
- **Rejected** (Red): Document needs resubmission
- **Real-Time Updates**: Status updates automatically

### Review Process
- **Admin Review**: Coaches/admins review all submissions
- **Feedback**: Rejection includes reason for resubmission
- **Resubmission**: Easy to upload corrected documents
- **History**: See all your past submissions

## Upload Tips

### Camera Photos

**Lighting:**
- Use bright, even lighting
- Avoid shadows on document
- Natural light works best
- Avoid glare from overhead lights

**Position:**
- Lay document flat on table
- Hold phone parallel to document (not at angle)
- Include entire document in frame
- Keep margins in view
- Make sure nothing is cut off

**Focus:**
- Tap on document to focus
- Wait for camera to focus
- Ensure text is sharp and clear
- Don't use zoomed photos if blurry

**Legibility:**
- All text must be readable
- Signatures clearly visible
- No portions obscured
- Check before submitting

### File Uploads

**File Format:**
- **PDF**: Best for scanned documents
- **JPG/JPEG**: Good for photos
- **PNG**: Also acceptable
- Check file extension before uploading

**File Size:**
- Most devices handle typical document sizes
- If file too large:
  - Reduce image quality slightly
  - Crop to just the document
  - Use compression tool
  - Take new photo at lower resolution

**File Names:**
- Use descriptive names
- Example: "2024_Registration_JohnSmith.pdf"
- Helps you identify documents later
- Not required but recommended

## Understanding Status Updates

### Pending Review
- **What It Means**: Document uploaded successfully, waiting for admin to review
- **How Long**: Usually reviewed within 1-3 business days
- **Your Action**: None needed, just wait
- **Notification**: May receive notification when reviewed

### Approved
- **What It Means**: Document accepted, requirements met
- **Your Action**: None needed
- **Record**: Keep confirmation for your records
- **Expiration**: Note if document expires annually

### Rejected
- **What It Means**: Document cannot be accepted as-is
- **Reason**: Check rejection notes from admin
- **Your Action**: Fix issue and resubmit
- **Timeline**: Resubmit as soon as possible
- **Help**: Contact admin if unclear why rejected

## Best Practices

### Submission Timing
- Submit documents as soon as requested
- Don't wait until deadline
- Allows time for resubmission if needed
- Early submission = less stress

### Quality Control
- Review photo before submitting
- Zoom in to check legibility
- Retake if any doubt about quality
- Better to spend extra minute than get rejected

### Organization
- Keep list of required documents
- Check off as you submit
- Save copies on your device
- Note expiration dates

### Communication
- Read rejection reasons carefully
- Contact admin if confused
- Ask questions before submitting if unsure
- Provide correct document type

## Privacy & Security

### Document Storage
- **Secure**: Documents stored in Firebase Cloud Storage
- **Encrypted**: Protected in transit and at rest
- **Private**: Only you and admins can view your documents
- **Compliance**: Follows data protection best practices

### Access Control
- **You**: Can view your own documents
- **Admins**: Can view, review, approve/reject documents
- **Other Users**: Cannot see your documents
- **Parents**: Can view documents for linked athletes

### Data Retention
- Documents stored for duration of membership
- Follow BPVA data retention policies
- Request deletion by contacting admin
- Backup important documents yourself

## Need Help?

### Common Questions

**Q: What if my photo is blurry?**
A: Retake the photo with better focus and lighting. You can resubmit as many times as needed.

**Q: How do I know what documents to submit?**
A: Check with coaches or admins for required documents list. Usually provided at registration.

**Q: Can I delete a document after uploading?**
A: Contact an admin if you need to remove a document. They can manage your submissions.

**Q: What if I don't have a PDF scanner?**
A: Use the camera feature in the app to photograph paper documents. Make sure photos are clear.

**Q: How long until my document is reviewed?**
A: Usually 1-3 business days. Contact admin if urgent or if waiting longer.

**Q: Can I submit before completing the form?**
A: No, complete all required fields and signatures before submitting.

### Contact Support
If you need help:
1. Check that document is clear and complete
2. Read any rejection reasons carefully
3. Try resubmitting with corrections
4. Contact coach or admin for questions
5. Provide screenshot of any error messages
