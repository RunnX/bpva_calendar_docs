# BPVA Calendar - User Documentation

Welcome to the BPVA Calendar user documentation! This guide provides comprehensive information for athletes and parents using the app.

## Getting Started

New to BPVA Calendar? Start here:

1. [**Sign In & Sign Out**](sign-in.md) - Authenticate with Google to access all features
2. [**My Profile**](profile.md) - Set up your profile and emergency contacts
3. [**Messages & Announcements**](messages.md) - Stay informed with coach communications

## Core Features

### For All Users

- [**Messages & Announcements**](messages.md)
  - Receive notifications from coaches
  - View urgent alerts and updates
  - Track read/unread messages

- [**Sign In & Sign Out**](sign-in.md)
  - Google authentication
  - Guest mode for calendar viewing
  - Account management

- [**My Profile**](profile.md)
  - Manage personal information
  - Update emergency contacts
  - Link family accounts (parents/athletes)

### For Athletes

- [**Gym Check-In**](gym-checkin.md)
  - Record attendance with GPS verification
  - Track training frequency
  - View check-in history

- [**Runway Step Configurations**](runway-steps.md)
  - Manage pole vault approach settings
  - Track pole specifications
  - Save multiple configurations

- [**Practice Journal**](journal.md)
  - Document training sessions
  - Record competition results
  - Export to PDF

- [**Documents & Forms**](documents.md)
  - Submit registration documents
  - Upload waivers digitally
  - Track submission status

## User Roles

The BPVA Calendar app provides different features based on your role:

### Athletes
Full access to:
- Calendar viewing
- Messages and announcements
- Gym check-in
- Runway step configurations
- Practice journal
- Document submission
- Profile with emergency contacts

### Parents
Access to:
- Calendar viewing
- Messages and announcements
- Linked athlete information
- Profile management
- Emergency contact updates

### Regular Users
Access to:
- Public calendar viewing
- Messages and announcements
- Basic profile
