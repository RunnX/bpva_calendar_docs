## üîê Security & Permissions

### Admin Role Requirements
- Only users with **Admin** role can access admin features
- Admins have full read/write access to all data
- Cannot remove the last admin (system protection)
- All admin actions are logged

### Best Practices
- Use strong passwords with Google/Apple Sign-In
- Don't share admin credentials
- Review admin list quarterly
- Remove admin access when staff leaves
- Follow least-privilege principle
